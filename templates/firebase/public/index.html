Firebase Project Template

This template demonstrates using Firebase Hosting, and is based on the One-File-Per-Type template. (To see Firebase Auth and Database examples, check out the React on Firebase template.)

Note: This example uses odigity's Firebase Project ID for deployment and hosting, which you don't have access to. In order to try deploying it yourself, first create your own Firebase Project, then substitute your own Project ID in '.firebaserc'. See the Firebase doc for more info.

You can see this app hosted on Firebase here: https://project-template-hosting.firebaseapp.com/

Template Structure Notes

  • public/ Folder — Notice our app files have now been moved into a 'public/' subdirectory. This is a better convention than putting your content directly in the project root. It also works with Firebase's convention of specifying the location of your public folder in 'firebase.json'. That folder will be served at the root of the hosted website, so 'public/index.html' will be served at https://project-template-hosting.firebaseapp.com/.
  • firebase.json — This is a JSON file that specifies your project's settings for the various Firebase services. In this case, we're only using Firebase Hosting.
  • .firebase.rc — This is also a JSON file that mostly is just for specifying the Project ID of the default project to use when deploying.
