React on Firebase Project Template

This template demonstrates using React with Firebase Hosting, Auth, and Database. It is based on a combination of the Client-Side React and Firebase templates.

See the Firebase and React docs for more info.

You can see this app hosted on Firebase here: https://project-template-react.firebaseio.com/

Template Structure Notes

  • database.rules.json — This is a new file that you can use store the Firebase Database Rules for your project. You also have to tell Firebase which file contains your database rules, which you can see specified in the "database" section of 'firebase.json'.
  • Rewrites — We've also added a "rewrites" section to 'firebase.json' that tells the Firebase Hosting servers that all requests should be redirected to index.html. That way when if a user clicks on the 'About' link to go to '/about' URL, then hits refresh, the app will be reloaded instead of returning a 404 Not Found.
